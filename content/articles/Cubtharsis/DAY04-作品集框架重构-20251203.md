### 2. 游戏设计方向重大调整 ⭐⭐⭐⭐⭐

从"关卡制解谜"转变为"开放世界探索+情绪能力解锁"：

**旧设计**：
- 线性关卡，一关一关推进
- 六种情绪，组合复杂

**新设计**：
- 开放世界自由探索（参考旷野之息）
- 四种情绪：悲伤、愤怒、恐惧、孤独
- 双层谜题结构（外层教学+内层箱庭）
- 魔方嵌入地面，转动改变世界地形

### 3. 涌现式情绪系统设计 ⭐⭐⭐⭐⭐

放弃"硬编码组合"，改用涌现式设计：

| 情绪 | 环境效果 |
|------|---------|
| **悲伤** | 下雨（产生水、湿润） |
| **愤怒** | 升温（产生热、干燥） |
| **恐惧** | 天色变暗/光源移动 |
| **孤独** | 时间回溯（创造过去的自己） |

情绪改变环境 → 环境影响物体 → 物体自然交互 = 涌现效果

### 4. 魔方地砖系统设计 ⭐⭐⭐⭐⭐

**核心概念**：魔方不是独立谜题空间，而是嵌入世界的地砖

```
魔方嵌在地面 → 浮起来 → 转动改变纹路连接 → 落回去 → 世界变了
```

**六种纹路类型**：
- 水渠（改变水流方向）
- 电路（改变电流路径）
- 光路（改变光线反射）
- 轨道（改变物体滚动）
- 藤蔓（改变植物生长）
- 时间线（改变过去的自己路径）

### 5. 隐藏复原设计 ⭐⭐⭐⭐⭐

**设计理念**：玩家不知道目标是复原魔方

每个谜题只告诉玩家"让水流到这里"或"让光照到那里"，但设计师暗中安排——每个谜题的解法刚好是复原魔方的一步。

玩家玩着玩着，突然发现魔方复原了 = 不知不觉中接纳了完整的自己

## 💡 关键突破

### 突破1：从六种情绪精简到四种

**问题**：六种情绪组合太复杂，设计不完

**解决**：精简到四种最核心的负面情绪
- 悲伤、愤怒、恐惧：心理学基本负面情绪
- 孤独：现代年轻人最普遍的情绪体验

### 突破2：涌现式设计替代硬编码组合

**问题**：11种组合效果需要逐个设计，工作量巨大

**解决**：参考旷野之息，设计环境属性系统
- 不设计"悲伤+愤怒=蒸汽"
- 而是设计"水+热=蒸汽"，让它们自然交互

### 突破3：魔方即地砖

**问题**：箱庭谜题和大世界割裂，缺乏联系

**解决**：魔方嵌入地面，转动直接改变世界地形
- 水渠连通 → 水流过去
- 电路连通 → 机关启动
- 光路调整 → 日晷触发

### 突破4：恐惧和孤独的涌现式设计

**问题**：恐惧和孤独如何产生涌现效果？

**解决**：
- 恐惧 = 改变光源/天色 → 影子方向变化 → 光影解谜
- 孤独 = 时间回溯 → 过去的自己 → 双人协作谜题（参考时空幻境）

| 维度 | 旧设计 | 新设计 |
|------|--------|--------|
| 世界结构 | 线性关卡 | 开放世界 |
| 情绪数量 | 6种 | 4种 |
| 组合方式 | 硬编码11种 | 涌现式自然交互 |
| 箱庭关系 | 独立谜题空间 | 嵌入世界的地砖 |
| 复原目标 | 明确告知 | 隐藏设计，自然发现 |
